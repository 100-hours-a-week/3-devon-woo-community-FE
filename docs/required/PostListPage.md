2) UI/UX 요소
▶ 게시글 작성 버튼
상태	스타일
기본	배경색 #ACA0EB
마우스 hover	#7F6AEE
hover 해제	다시 #ACA0EB

클릭 → 게시글 작성 페이지 이동

▶ 게시글 리스트 카드 구성
항목	요구사항
제목	최대 26자, 초과 시 잘라서 표시
날짜/시간	yyyy-MM-dd hh:mm:ss
카운트 표시	조회수, 댓글수, 좋아요수
카운트 단위	1,000→1k / 10,000→10k / 100,000→100k

예:

1,234 → 1.2k

52,000 → 52k
