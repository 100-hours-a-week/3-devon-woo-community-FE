# 프로필 페이지 디자인

## 개요
Tech Blog 스타일의 프로필 페이지로, 사용자의 정보와 작성한 글을 한눈에 보여주는 페이지입니다.
Tech Blog 메인 페이지의 레이아웃을 참고하되, 프로필에 최적화된 구조로 설계합니다.

## 레이아웃 구조

```
+----------------------------------+------------------------------------------+
|                                  |                                          |
|  프로필 카드 (왼쪽)               |  작성 글 목록 (오른쪽)                      |
|  - 고정 너비 (300-350px)          |  - 유동 너비 (나머지 공간)                   |
|                                  |                                          |
|  [프로필 사진]                     |  [최신순 ▼]                               |
|                                  |                                          |
|  사용자 이름                       |  +------------------------------------+  |
|  @username                       |  | Understanding React Hooks          |  |
|  간단한 소개                       |  | April 10, 2024                     |  |
|                                  |  | React Hooks evthe change...         |  |
|  통계                             |  | [React] [JavaScript] [Web Dev]      |  |
|  - 작성 글: 315                   |  +------------------------------------+  |
|  - 팔로워: 1                      |                                          |
|  - 팔로잉: 0                      |  +------------------------------------+  |
|                                  |  | A Guide to Python Generators       |  |
|  [프로필 편집] (본인만)             |  | March 15, 2024                     |  |
|                                  |  | Python's generators in an...        |  |
|  카테고리                          |  | [Python] [Programming]              |  |
|  - Web Development               |  +------------------------------------+  |
|  - Programming                   |                                          |
|  - Data Science                  |  [1] [2] [Next]                          |
|  - DevOps                        |                                          |
|                                  |                                          |
|  태그                             |                                          |
|  [JavaScript] [React]             |                                          |
|  [Python] [Docker] [CSS]          |                                          |
|                                  |                                          |

```

## 주요 섹션

### 1. 헤더 (Header)
- Tech Blog와 동일한 헤더 유지
- 로고, 네비게이션 메뉴, 검색, 다크모드 토글

### 2. 왼쪽 프로필 카드 (Profile Card)

#### 2.1 프로필 기본 정보
- **프로필 사진**: 원형, 160x160px
- **사용자 이름**: 큰 폰트, 볼드
- **핸들/레벨**: 작은 폰트, 회색 (@username 또는 Lv.0 베이비)
- **간단한 소개**: 2-3줄, 사용자가 입력한 자기소개

#### 2.2 통계 정보
```
+----------+----------+----------+
|   315    |    1     |    0     |
|   작성    |  팔로워   |  팔로잉   |
+----------+----------+----------+
```

#### 2.3 프로필 편집 버튼
- 본인 프로필일 경우만 표시
- 전체 너비, 회색 배경

#### 2.4 카테고리 (Categories)
- 사용자가 글을 작성한 카테고리 목록
- 클릭 시 해당 카테고리 글 필터링

#### 2.5 태그 (Tags)
- 사용자가 자주 사용한 태그
- 클릭 시 해당 태그 글 필터링

### 3. 오른쪽 작성 글 목록 (Posts List)

#### 3.1 정렬 옵션
```
[최신순 ▼]  [인기순]  [조회순]
```

#### 3.2 포스트 카드
각 포스트는 Tech Blog 메인과 동일한 스타일:
- 제목: 큰 폰트, 볼드
- 날짜: 작은 폰트, 회색
- 요약: 2-3줄 미리보기
- 태그: 회색 배경 뱃지
- 호버 효과: 살짝 올라가는 애니메이션

#### 3.3 페이지네이션
- 하단에 페이지 번호
- [1] [2] [Next] 형식

## 반응형 디자인

### 데스크톱 (1024px 이상)
- 왼쪽 프로필: 350px 고정
- 오른쪽 글 목록: 나머지 공간

### 태블릿 (768px - 1023px)
- 왼쪽 프로필: 300px 고정
- 오른쪽 글 목록: 나머지 공간
- 포스트 카드 세로 정렬

### 모바일 (767px 이하)
- 프로필 카드가 상단에 배치
- 글 목록이 하단에 세로로 배치
- 단일 컬럼 레이아웃

## 인터랙션

### 프로필 카드
- 카테고리 클릭 → 해당 카테고리 글 필터링
- 태그 클릭 → 해당 태그 글 필터링
- 통계 클릭 → 상세 정보 모달 (선택사항)

### 포스트 카드
- 카드 호버 → 살짝 올라가는 애니메이션 + 그림자
- 카드 클릭 → 포스트 상세 페이지로 이동
- 태그 클릭 → 해당 태그 글 필터링

### 정렬
- 정렬 옵션 클릭 → 글 목록 재정렬
- 부드러운 트랜지션 효과

## 참고 디자인

### Tech Blog 메인 페이지
- 왼쪽 사이드바 → 프로필 카드로 변환
- 오른쪽 포스트 목록 → 동일하게 유지

### 라프텔 프로필 페이지
- 프로필 정보 레이아웃 참고
- 통계 섹션 스타일 참고
- 탭 구조는 Tech Blog 스타일로 단순화

## 구현 우선순위

1. HTML 구조 작성
2. 프로필 카드 스타일링
3. 포스트 목록 스타일링
4. 반응형 레이아웃
5. 다크 모드
6. 인터랙션 및 애니메이션
7. 필터링 기능
