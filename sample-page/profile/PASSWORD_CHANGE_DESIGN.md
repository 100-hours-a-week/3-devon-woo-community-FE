# 비밀번호 변경 페이지 디자인

## 개요
사용자가 비밀번호를 안전하게 변경할 수 있는 페이지입니다.

## 레이아웃
- profile-edit.html과 동일한 중앙 카드 레이아웃
- max-width: 540px (더 작은 폼)
- 간단한 3개 입력 필드 + 버튼

## 폼 구조

### 입력 필드
1. **현재 비밀번호** (필수)
   - 보안을 위해 현재 비밀번호 확인

2. **새 비밀번호** (필수)
   - 최소 8자 이상
   - 영문, 숫자, 특수문자 조합 권장

3. **새 비밀번호 확인** (필수)
   - 새 비밀번호와 일치해야 함

### 기능
- 비밀번호 표시/숨기기 토글
- 실시간 비밀번호 강도 표시
- 비밀번호 일치 여부 검증
- 저장/취소 버튼

## 검증 규칙
- 현재 비밀번호: 필수
- 새 비밀번호: 8자 이상
- 새 비밀번호 확인: 새 비밀번호와 일치
- 현재 비밀번호와 새 비밀번호 달라야 함
